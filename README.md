# handwriting-to-obsidian

## Setup Instructions

1. Create your own Obsidian test vault. `git clone` this repository into `.obsidian/plugins/`.
2. `git clone` the Hot Reload plugin: `git clone https://github.com/pjeby/hot-reload`
3. `pnpm install` necessary packages.

Use `pnpm watch` to auto build whenever you make changes to the source files.
